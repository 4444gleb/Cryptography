Подписывание исполняемого файла

Для создания самоподписанного сертфиката для подписывания кода приложений выполняем команду в Power Shell:
$cert = New-SelfSignedCertificate -Subject "Certificate Samosledov” -Type CodeSigningCert -CertStoreLocation cert:\LocalMachine\My

Далее экспортируем сертификат.
Выполнить -> mmc.exe

В открывшейся консоли выбираем:
Файл -> добавить или удалить оснастку -> Сертификаты -> Добавить

Находим наш сертификат (можно воспользоваться поиском), ПКМ по сертификату -> Экспорт. 

Мастер установки - Да, экспортировать закрытый ключ - Далее - (указываем пароль) - (выбираем директорию)

Переходим в директорию, где лежит программа SignTool:
cd "C:\Program Files\Microsoft SDKs\Windows\v6.0A\bin"

Все готово, можно подписывать
.\signtool sign /f D:\Samosledov.pfx /p "superhot" C:\IDEA.exe
